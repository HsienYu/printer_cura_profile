Cura profiles for the Original Prusa i3
This an official set of settings from Prusa Research for Cura software by Ultimaker.

Installation guide:
Cura 3.5.1 and newer https://manual.prusa3d.com/Guide/How+to+import+profiles+to+Cura/703
Cura 3.4.1 and older https://manual.prusa3d.com/Guide/How+to+import+profiles+to+Cura/1033
